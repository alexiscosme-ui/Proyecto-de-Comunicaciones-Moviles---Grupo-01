# 📡 Implementación de una Red Privada LTE con soporte para NB-IoT mediante SDR

## 📌 Descripción
Este proyecto tiene como objetivo la implementación de una red privada LTE con soporte para NB-IoT utilizando radio definido por software (SDR). Se busca analizar el funcionamiento de la red, así como evaluar métricas de desempeño como latencia y throughput.

## 🎯 Objetivos
- Implementar una estación base LTE usando srsRAN.
- Configurar el núcleo de red (EPC).
- Integrar hardware SDR (USRP).
- Evaluar el desempeño de la red en un escenario IoT.

## 🛠️ Tecnologías utilizadas
- srsRAN 4G
- USRP (SDR)
- Ubuntu Linux
- NB-IoT
